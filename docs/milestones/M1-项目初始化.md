## ⚠️ 注意事项

1. **Next.js 16 有破坏性变更**：先阅读 `node_modules/next/dist/docs/`
2. **Client Component 标记**：使用 `usePathname` 的组件需加 `'use client'`
3. **路由分组不影响 URL**：`(platform)` 不会出现在 URL 中
4. **根页面重定向**：在 `app/page.tsx` 使用 `redirect('/chat')`
