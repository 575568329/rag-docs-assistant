# 向量检索算法速查表

> 向量检索核心算法和参数

## 余弦相似度

### 公式
```
similarity = dot(a, b) / (||a|| × ||b||)

其中:
- dot(a, b) = a₁×b₁ + a₂×b₂ + ... + aₙ×bₙ
- ||a|| = √(a₁² + a₂² + ... + aₙ²)
```

### 范围
- [-1, 1]
- 1 表示完全相同
- 0 表示正交（无关）
- -1 表示完全相反

### TypeScript 实现
```typescript
function cosineSimilarity(a: number[], b: number[]): number {
  const dotProduct = a.reduce((sum, val, i) => sum + val * b[i], 0);
  const normA = Math.sqrt(a.reduce((sum, val) => sum + val * val, 0));
  const normB = Math.sqrt(b.reduce((sum, val) => sum + val * val, 0));
  return dotProduct / (normA * normB);
}
```

## RRF（Reciprocal Rank Fusion）

### 公式
```
score = Σ(1 / (k + rank_i))

其中:
- k = 60 (常数，避免前排差距过大)
- rank_i = 该结果在第 i 个排序列表中的排名
```

### 用途
融合多个排序列表（向量搜索 + 关键词搜索）

### TypeScript 实现
```typescript
function rrf(
  rankings: Array<{idx: number, rank: number}[]>,
  k = 60
): Map<number, number> {
  const scores = new Map<number, number>();
  
  rankings.forEach(ranking => {
    ranking.forEach(({idx, rank}) => {
      const score = 1 / (k + rank);
      scores.set(idx, (scores.get(idx) || 0) + score);
    });
  });
  
  return scores;
}
```

## TF-IDF

### 公式
```
TF-IDF = TF × IDF

TF (词频) = 词在文档中出现次数 / 文档总词数
IDF (逆文档频率) = log(总文档数 / 包含该词的文档数)
```

### 特点
- 高频词（"的"、"是"）的 IDF 低
- 罕见词的 IDF 高
- 用于关键词搜索排序

## 核心参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **相似度阈值** | 0.45 | 低于此值的结果被过滤 |
| **Top-K** | 5 | 返回前 K 条结果 |
| **RRF k 值** | 60 | 融合算法常数 |
| **切片大小** | 1000 字符 | 文本切片单位 |
| **切片重叠** | 100 字符 | 保持上下文连续性 |

---

## 混合搜索流程

```
1. 向量搜索
   ├─ 计算查询向量与所有文档的余弦相似度
   └─ 排序得到 vectorRanking

2. 关键词搜索
   ├─ 分词
   ├─ 计算 TF-IDF
   └─ 排序得到 keywordRanking

3. RRF 融合
   ├─ 融合两个排序列表
   └─ 返回 Top-K
```

---

## ⚠️ 注意事项

1. **归一化**：向量相似度计算前确保向量已归一化
2. **阈值调整**：0.45 是经验值，需根据实际效果调整
3. **性能优化**：大规模向量搜索需要 HNSW、IVF 等索引
4. **中文分词**：简单方案按字拆分，精确方案使用分词库
